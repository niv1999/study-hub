חלק 2 — שאלות פתוחות (כתיבת SQL)

נסו לכתוב את השאילתה בעצמכם, ואז גלו את הפתרון לדוגמה.

שאלה פתוחה 1

כתבו שאילתה המחזירה את שם המחלקה ואת מספר העובדים בכל מחלקה, ממוין ממחלקה עם הכי הרבה עובדים.

SELECT d.dept_name, COUNT(e.emp_id) AS num_employees
FROM department d
LEFT JOIN employee e ON e.dept_id = d.dept_id
GROUP BY d.dept_id, d.dept_name
ORDER BY num_employees DESC;

LEFT JOIN כדי לכלול גם מחלקות ללא עובדים (יופיעו עם 0). מקבצים לפי מחלקה, סופרים עובדים וממיינים יורד.

שאלה פתוחה 2

כתבו את פקודת ה-DDL ליצירת טבלת department: dept_id מפתח ראשי, dept_name טקסט חובה (עד 50 תווים), location טקסט.

CREATE TABLE department (
  dept_id   INT PRIMARY KEY,
  dept_name VARCHAR(50) NOT NULL,
  location  VARCHAR(50)
);

PRIMARY KEY מזהה כל שורה; NOT NULL הופך את שם המחלקה לחובה; VARCHAR(50) מגביל את אורך הטקסט.
