מבוא לווב וענן

אתר לימוד עצמי מרוכז לקורס מבוא לווב וענן (שנקר, הנדסת תוכנה — המרצה יובל עוזרי). המסלול בונה אפליקציית ווב מהיסוד: HTML (מבנה תקני, תגיות סמנטיות), CSS (סלקטורים, Box Model, Flexbox ורספונסיביות), JavaScript ונילה (טיפוסים, פונקציות, מערכים, DOM ואירועים), ומשם למודל לקוח–שרת (HTTP, fetch/AJAX, JSON, טפסים ו-Bootstrap), צד שרת עם Node.js, npm ו-Express, ולבסוף Full-Stack עם MongoDB/Mongoose, Git/GitHub ופריסה לענן ב-Render. לצד הסיכומים — עמוד מלכודות ממוקד וחמישה מבחני תרגול אמריקאיים בקושי עולה.
